#### 静态链表

```c
#include <stdio.h>
#include <cstdlib>
struct Node
{
    int data;
    struct Node *next;
};

int main()
{
    struct Node node1 = {2, NULL};
    struct Node node2 = {3, NULL};
    struct Node node3 = {4, NULL};
    struct Node *P;

    node1.next = &node2; //将note2的地址放到note1的next中
    node2.next = &node3;//将note3的地址放到note2的next中
    /*
    现在node1中的data为2，next为2的地址
    node1中的data为2，next为2的地址
    */
    printf("%d", node1.next);
    system("pause");
    return 0;
}
```

解释：

1. 首先定义了一个结构体
2. 在主函数中定义了三个结构体变量
3. 将三个结构体变量连接起来
